Les métadonnées EXIF

Chaque photo numérique contient des informations cachées : la date, l'heure, les réglages de l'appareil, et parfois votre position GPS. Ces données s'appellent les métadonnées EXIF.

Qu'est-ce que les métadonnées EXIF ?

EXIF est l'abréviation de Exchangeable Image File Format. Il s'agit d'un standard qui définit la manière dont les appareils photo numériques stockent des informations techniques à l'intérieur même des fichiers image, sans que cela ne soit visible à l'œil nu.

Ces informations, appelées métadonnées (des données qui décrivent d'autres données), sont automatiquement enregistrées par l'appareil photo au moment de la prise de vue. On y trouve par exemple la marque et le modèle de l'appareil, la focale utilisée, la vitesse d'obturation, l'ouverture, la sensibilité ISO, la date et l'heure, et parfois même les coordonnées GPS du lieu de prise de vue.

Le standard EXIF a été créé par l'association japonaise de l'industrie électronique (JEITA) et est supporté par la majorité des appareils photo numériques et smartphones depuis les années 2000. Il fonctionne avec les formats JPEG, TIFF et RAW, mais pas avec le PNG ou le GIF.

Attention à la vie privée : si vous publiez des photos sur internet, les métadonnées EXIF peuvent révéler votre localisation GPS ou d'autres informations personnelles. La plupart des réseaux sociaux les suppriment automatiquement, mais il vaut mieux les effacer soi-même au préalable.

Comment lire les métadonnées EXIF d'une photo ?

Plusieurs méthodes permettent de consulter les données EXIF d'une image :

  1. Sur Windows : clic droit sur le fichier image → « Propriétés » → onglet « Détails ». Les données EXIF apparaissent dans la liste.
  2. Sur macOS : ouvrir l'image dans Aperçu → menu « Outils » → « Afficher l'inspecteur » → onglet avec l'appareil photo.
  3. Dans les logiciels photo : Adobe Lightroom, GIMP, FastStone Image Viewer affichent tous les EXIF dans un panneau dédié.
  4. En ligne : des sites comme Jeffrey's Exif Viewer ou ExifData.com permettent de lire les EXIF sans installer de logiciel.
  5. Via la ligne de commande : l'outil libre ExifTool (par Phil Harvey) est une référence pour lire, modifier ou supprimer des métadonnées en masse.
  6. Sur smartphone : l'application Google Photos affiche les détails EXIF dans les informations de la photo (icône « i »).

À quoi servent les métadonnées EXIF ?

Les données EXIF sont utiles dans de nombreuses situations :

  • Apprentissage : un photographe peut analyser ses propres réglages pour progresser.
  • Organisation : les logiciels de gestion classent automatiquement les photos par date ou lieu.
  • Géolocalisation : on peut retrouver sur une carte l'endroit exact où une photo a été prise.
  • Journalisme : vérifier la date et le lieu d'une photo pour confirmer son authenticité.
  • Droit d'auteur : certaines données EXIF permettent d'identifier le photographe.
  • Correction automatique : les logiciels peuvent corriger la distorsion de l'objectif grâce aux EXIF.
  • Forensique numérique : les enquêteurs utilisent les EXIF pour dater ou localiser des prises de vue.

Comment supprimer les métadonnées EXIF ?

Avant de partager une photo sur internet, il peut être prudent de supprimer les données EXIF sensibles, en particulier la localisation GPS. Voici comment procéder :

Sur Windows :
Clic droit → Propriétés → Détails → « Supprimer les propriétés et informations personnelles » → Cocher les éléments à supprimer → OK.
Avec ExifTool :
Commande : exiftool -all= maphoto.jpg Supprime toutes les métadonnées en une seule instruction.

Des applications mobiles comme Photo Investigator (iOS) ou Exif Eraser (Android) permettent aussi de gérer facilement les métadonnées directement depuis un smartphone.
